Claude Chat（MCPコネクタ）か Claude Code をこの mio-memory につないだら、こう聞くだけ：

> 「mio-memory の使い方を教えて」

## 最初にやること（これだけ）

1. **`core_stable.md`** … あなたのアシスタントは誰か（名前・大事にすること・あなたとの関係）。`<...>` を埋める
2. **`core_infra.md`** … サーバURL・バージョン等の環境情報。`<...>` を埋める
3. あとは**対話しながら育てていけばOK**。記憶もルールも、使いながら増やせます

> 埋めるのは admin.html の CoreMem タブ、または Claude に「core_stable.md をこう書き換えて」と頼めばOK。
